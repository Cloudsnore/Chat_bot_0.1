# Chat Bot Antigravity

A mental wellness AI chatbot application built with Next.js. This app provides conversational support with integrated crisis detection and escalation mechanisms to ensure user safety.

## Features

- Polished chat interface with real-time streaming responses
- Animated interactions for enhanced user experience
- Crisis detection with modal system for handling emergency situations
- Built-in safety mechanisms and content filtering

## Tech Stack

- **Framework**: Next.js 16.2.1 with React 19.2.4
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4 with shadcn/ui components
- **Animations**: Framer Motion 12.38.0
- **AI Integration**: AI SDK 6.0.138 with Google Gemini (gemini-1.5-flash)
- **Development Tools**: ESLint 9, PostCSS

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to start chatting with the bot.
